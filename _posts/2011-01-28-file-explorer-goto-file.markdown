在文件浏览器中快速定位到一个文件的方法：    

- windows中*我的电脑*，或者*资源管理器*
    - 要定位一个文件的话，输入文件的首字母。反复按一个字母，就在文件名为这个字母的文件间转换。
    - 如果快速连续输入的话，就可用文件名的前几个字母定位到。
- Linux中文件管理器。kde中的konqueror
    - 连续输入文件名，但是不能**“反复按一个字母，就在文件名为这个字母的文件间转换。”**
    - 如果要在输入字母开头的文件间转换——只要向上或向下滚动鼠标中建，或者按向上向下间就可以了
